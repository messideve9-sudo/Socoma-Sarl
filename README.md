# Suivi des Créances - Bouillon Kadi-Jadida Section Marchés

## 📋 Description
Application web de suivi des créances conçue pour la section marchés de Bouillon Kadi-Jadida. Le système permet de relancer efficacement les clients et d'avoir une visibilité complète sur le portefeuil client de l'entreprise.

## 🎯 Objectifs
- Centraliser le suivi des créances clients
- Automatiser les relances clients
- Visualiser en temps réel le portefeuil client
- Générer des rapports analytiques
- Améliorer le recouvrement des créances

## ✨ Fonctionnalités principales
- **Gestion des clients** : Ajout, modification et archivage
- **Suivi des paiements** : Historique complet des transactions
- **Système de relance** : Alertes et notifications automatiques
- **Tableaux de bord** : Visualisations graphiques des données
- **Rapports** : Génération de rapports PDF/Excel
- **Sécurité** : Accès protégé par authentification

## 🛠 Technologies utilisées
- **Backend** : Python 3.11.4
- **Framework** : Flask / Django
- **Base de données** : PostgreSQL / SQLite
- **Frontend** : HTML, CSS, JavaScript
- **Déploiement** : Render.com
- **Versioning** : Git & GitHub

## 🚀 Installation locale

### Prérequis
- Python 3.11.4
- pip (gestionnaire de packages)

### Étapes d'installation
```bash
# 1. Cloner le dépôt
git clone https://github.com/votre-utilisateur/suivi-creances.git

# 2. Accéder au dossier
cd suivi-creances

# 3. Installer les dépendances
pip install -r requirements.txt

# 4. Lancer l'application

python app.py
